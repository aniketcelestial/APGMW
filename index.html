I'm a final-year Mathematics, Statistics and Computer Science student at BHU, Varanasi, deeply fascinated by how software shapes human experience. I write code with the belief that every line should serve a purpose — functional, efficient, and readable.

My journey started with a curious Google search about "how websites work" at age 15. Since then, I've gone from building static HTML pages to crafting full-stack applications, exploring machine learning models, exploring system designing with low level programming and contributing to open-source.
